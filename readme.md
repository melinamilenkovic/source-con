# Fitly

Fitly ist eine einfache Webanwendung zum erfassen und verwalten von Trainingseinheiten.
Nutzer können ihre Workouts dokumentieren, Fortschritte verfolgen und ihre persönliche Trainingshistorie übersichtlich einsehen.

## Features

- Liste anzeigen

## Technologien

- HTML
- CSS
- JavaScript
- `localstorage` für die Datenspeicherung im Browser

## Projektstruktur

- `readme.de`
- `index.html`
- `styles.css`

## Datenverwaltung

Für jedes Workout können folgende Informationen gespeihert werden:

- Trainingsart z.B. Krafttraining, Cardio
- Datum des Trainings
- Übungen
- Gewicht/Wiederholungen/Dauer
- Optionaler Kommentar zum Training

## Idee hinter dem Projekt

Fitly soll als digitales Trainingstagebuch dienen.
Ziel ist es, Fortschritte sichtbar zu machen, Training zu strukturieren und die eigene Entwicklung langfristig zu verbessern.

## Ausblick

- Backend-Anbindung mit Datenbank
- Workouts hinzufügen
- Trainingsdaten bearbeiten
- Workouts löschen
- Speicherung der Daten im Browser über `localstorage`
